# Exhaustive-Search-Algorithms
Exhaustive Search Algorithms

Overview

This project was developed as part of an Algorithms course to study the Exhaustive Search (Brute Force) paradigm. It explains the concept of exhaustive search, its advantages and limitations, and analyzes common applications using algorithm analysis techniques.

Topics Covered

Linear Search
Finding the Minimum Element
Traveling Salesman Problem (TSP)
Time Complexity Analysis
Recurrence Relations
Substitution Method
